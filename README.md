## 简介

ToonHub 是一个基于 React + TypeScript + Vite 构建的前端漫画阅读应用。

## 技术栈

- 框架: React 19 + TypeScript
- 构建: Vite
- 路由: React Router
- 状态: React Hooks

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm run preview
```
